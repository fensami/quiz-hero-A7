Different between Map, ForEach, Filter, Find

Map

The map method creates a new array with the results of calling a provided function on every element in the calling array

ForEach

The forEach method calls a function for each element in an array. The forEach method is not executed for empty elements.

Filter

Filter method calls a function which elements is need to included according order

Find

Find method calls a function which element in the first item that element show us

Event loop

An event loop is an endless loop, which waits for tasks, executes them, and then sleeps until it receives more tasks. The event loop allows us to use callbacks and promises.
